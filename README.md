# C# Console Contact Manager & Logic Tools

This project demonstrates core **Object-Oriented Programming (OOP)** principles and robust **Exception Handling** in C# .NET 8.

## 🚀 Key Features
- **Contact Management:** A console system to add, modify, and delete records.
- **Data Validation:** Advanced string manipulation (Auto-Capitalization and Uppercase formatting).
- **Error Resilience:** Comprehensive `try-catch-finally` blocks to prevent crashes during user input.
- **Git Version Control:** Developed using a structured commit history.

## 🛠️ Built With
- **Language:** C#
- **Framework:** .NET 8.0
- **Environment:** Linux Mint / VS Code
